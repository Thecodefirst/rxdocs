                  TIP

                  RxComponents 内置Element-plus 版本为2.1.9

                  RxComponents 可以在支持 ES2018open in new window 和 ResizeObserveropen in new window 的浏览器上运行。 如果您确实需要支持旧版本的浏览器，请自行添加 Babelopen in new window 和相应的 Polyfill 。

                  # 使用包管理器

                  # 我们建议您使用包管理器（如 NPM、Yarnopen in new window 或 pnpmopen in new window）安装 RxComponents，然后您就可以使用打包工具，例如 Viteopen in new window 或 webpackopen in new window。

                  # 选择一个你喜欢的包管理器

# NPM
$ npm install rx-compo --save

# Yarn
$ yarn add rx-compo

# pnpm
$ pnpm install rx-compo

                  如果您的网络环境不好，建议使用相关镜像服务 cnpmopen in new window 或 中国 NPMopen in new window 镜像。

                  # unpkg

                  <head>
  <!-- Import style -->
  <link rel="stylesheet" href="//unpkg.com/rx-compo/dist/index.css" />
  <!-- Import Vue 3 -->
  <script src="//unpkg.com/vue@3"></script>
  <!-- Import component library -->
  <script src="//unpkg.com/rx-compo"></script>
</head>

                  # jsDelivr

                  <head>
  <!-- Import style -->
  <link
    rel="stylesheet"
    href="//cdn.jsdelivr.net/npm/rx-compo/dist/index.css"
  />
  <!-- Import Vue 3 -->
  <script src="//cdn.jsdelivr.net/npm/vue@3"></script>
  <!-- Import component library -->
  <script src="//cdn.jsdelivr.net/npm/rx-compo"></script>
</head>

                  TIP

                  我们建议使用 CDN 引入 RxComponents 的用户在链接地址上锁定版本，以免将来 RxComponents 升级时受到非兼容性更新的影响。 锁定版本的方法请查看 unpkg.comopen in new window。

                  # 用法

                  # 完整引入

                  如果你对打包后的文件大小不是很在乎，那么使用完整导入会更方便。

                  import { createApp } from 'vue'
import RxComponents from 'rx-compo/components'
import 'rx-compo/theme-chalk/src/index.scss'
import App from './App.vue'

createApp(App).use(RxComponents).mount('#app')

                  # Volar 支持

                  如果您使用 Volar，请在 tsconfig.json 中通过 compilerOptions.type 指定全局组件类型。

                  
// tsconfig.json
{
  "compilerOptions": {
    // ...
    "types": ["RxComponents/global"]
  }
}

                  # 按需导入 推荐

                  
import { createApp } from 'vue'
import { 
  RxButton,
  RxSelect,
  RxSelectPlus,
  RxOption,
  RxTooltip,
  RxInput,
  RxCheckbox
  } from 'rx-compo/components'
import 'rx-compo/theme-chalk/src/index.scss'
import App from './App.vue'

createApp(App)
  .use(RxButton)
  .use(RxSelect)
  .use(RxSelectPlus)
  .use(RxOption)
  .use(RxTooltip)
  .use(RxInput)
  .use(RxCheckbox)
  .mount('#app')

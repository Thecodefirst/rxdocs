                                  # RxRadio 在一组备选项中进行单选。

                                  # 基础用法

                                  单选框不应该有太多的可选项， 如果你有很多的可选项你应该使用选择框而不是单选框。

                                  要使用 RxRadio 组件，只需要设置v-model绑定变量， 选中意味着变量的值为相应RxRadio label属性的值， label可以是String、Number 或 Boolean。
                                  radio-label1
                                  radio-label2
                                  radio-label3
                                  基础 demo
                                  <template>
  <el-card shadow="never">
    <div>
      <rx-radio-group style="display: block;" v-model="radio1" :gutter="10">
        <el-row :gutter="10">
          <el-col :span="4" v-for="(item, index) in radioOptions" :key="index" >
            <rx-radio :label="item.label" size="large">{{ item.label }}</rx-radio>
          </el-col>
        </el-row>
      </rx-radio-group>
    </div>
    <div>
      <rx-radio-group style="display: block;" :gutter="10" :baseColProps="{ span: 4 }" v-model="radio2" :rxRadioGroupOptions="radioOptions">
      </rx-radio-group>
    </div>
  </el-card>
</template>
<script lang="ts" setup>
import { ref } from 'vue'

const radio1 = ref('radio-label1')
const radio2 = ref('radio-label1')

const radioOptions = ref([
  {
    label: 'radio-label1'
  },
  {
    label: 'radio-label2'
  },
  {
    label: 'radio-label3'
  }
])
</script>

                                  # 禁用状态

                                  disabled属性可以用来控制单选框的禁用状态。

                                  你只需要为单选框设置disabled属性就能控制其禁用状态。

                                  Option AOption B
                                  禁用 demo
                                  <template>
  <el-card shadow="never">
    <rx-radio v-model="radio" disabled label="disabled">Option A</rx-radio>
    <rx-radio v-model="radio" disabled label="selected and disabled"
      >Option B</rx-radio
    >
  </el-card>
</template>

<script lang="ts" setup>
import { ref } from 'vue'

const radio = ref('selected and disabled')
</script>

                                  # 带有边框

                                  设置border属性可以渲染为带有边框的多选框。
                                  边框 demo
                                  <template>
  <el-space>
    <rx-checkbox v-model="checked1" label="Option1" size="large" border ></rx-checkbox>
    <rx-checkbox v-model="checked2" label="Option2" size="large" border ></rx-checkbox>
    <rx-checkbox v-model="checked3" label="Option1" border ></rx-checkbox>
    <rx-checkbox v-model="checked4" label="Option2" border ></rx-checkbox>
  </el-space>
</template>

<script lang="ts" setup>
import { ref } from 'vue'

const checked1 = ref(true)
const checked2 = ref(false)
const checked3 = ref(false)
const checked4 = ref(true)
</script>

                                  # 单选框组

                                  适用于在多个互斥的选项中选择的场景

                                  结合rx-radio-group元素和子元素rx-radio可以实现单选组， 为 rx-radio-group 绑定 v-model，再为 每一个 rx-radio 设置好 label 属性即可， 另外，还可以通过 change 事件来响应变化，它会传入一个参数 value 来表示改变之后的值。

                                  Option AOption BOption C
                                  单选框组 demo
                                  <template>
  <el-card shadow="never">
    <rx-radio-group v-model="radio">
      <rx-radio :label="3">Option A</rx-radio>
      <rx-radio :label="6">Option B</rx-radio>
      <rx-radio :label="9">Option C</rx-radio>
    </rx-radio-group>
  </el-card>
</template>

<script lang="ts" setup>
import { ref } from 'vue'

const radio = ref(3)
</script>

                                  # 按钮样式

                                  按钮样式的多选组合。

                                  只需要把 rx-checkbox 元素替换为 rx-checkbox-button 元素即可。 此外，还提供了size属性。
                                  按钮样式 demo
                                  <template>
  <el-card shadow="never">
    <div>
      <rx-radio-group v-model="radio1" size="large">
        <rx-radio-button label="New York" ></rx-radio-button>
        <rx-radio-button label="Washington" ></rx-radio-button>
        <rx-radio-button label="Los Angeles" ></rx-radio-button>
        <rx-radio-button label="Chicago" ></rx-radio-button>
      </rx-radio-group>
    </div>
    <div style="margin-top: 20px">
      <rx-radio-group v-model="radio2">
        <rx-radio-button label="New York" ></rx-radio-button>
        <rx-radio-button label="Washington" ></rx-radio-button>
        <rx-radio-button label="Los Angeles" ></rx-radio-button>
        <rx-radio-button label="Chicago" ></rx-radio-button>
      </rx-radio-group>
    </div>
    <div style="margin-top: 20px">
      <rx-radio-group v-model="radio3" size="small">
        <rx-radio-button label="New York" ></rx-radio-button>
        <rx-radio-button label="Washington" disabled ></rx-radio-button>
        <rx-radio-button label="Los Angeles" ></rx-radio-button>
        <rx-radio-button label="Chicago" ></rx-radio-button>
      </rx-radio-group>
    </div>
  </el-card>
</template>

<script lang="ts" setup>
import { ref } from 'vue'

const radio1 = ref('New York')
const radio2 = ref('New York')
const radio3 = ref('New York')
</script>

                                  # 插槽

                                  支持函数式插槽slots、模板插槽template

                                  TIP

                                  默认插槽为template模板插槽
                                  函数式插槽slots渲染优先级大于模板插槽template

                                  模板插槽Label
                                  插槽 demo
                                  <template>
  <el-card shadow="never">
    <rx-radio-group v-model="radio">
      <!-- 模板插槽 -->
      <rx-radio label="模板插槽" size="large">
        <!-- $slots.default -->
        <div>模板插槽Label</div>
      </rx-radio>
      <!-- 函数式插槽 -->
      <rx-radio label="函数式插槽" size="large" :slots="radioSlots" ></rx-radio>
    </rx-radio-group>
  </el-card>
</template>

<script lang="ts" setup>
import { ref, h } from 'vue'
const radio = ref('模板插槽')
const radioSlots = () => h('div', {}, '函数式插槽Label')
</script>

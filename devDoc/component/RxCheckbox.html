                                # RxCheckbox 在一组备选项中进行多选。

                                # 基础用法

                                RxCheckboxGroup元素能把多个 RxCheckbox 管理为一组，只需要在 Group 中使用 v-model 绑定 Array 类型的变量即可。 只有一个选项时的默认值类型为 Boolean，当选中时值为true。 rx-checkbox 标签中的内容将成为复选框按钮之后的描述。
                                基础 demo
                                <template>
  <el-card shadow="never">
    <div>
      <rx-checkbox v-model="checked1" label="Option 1" size="large" ></rx-checkbox>
      <rx-checkbox v-model="checked2" label="Option 2" size="large" ></rx-checkbox>
    </div>
    <div>
      <rx-checkbox v-model="checked3" label="Option 1" ></rx-checkbox>
      <rx-checkbox v-model="checked4" label="Option 2" ></rx-checkbox>
    </div>
    <div>
      <rx-checkbox v-model="checked5" label="Option 1" size="small" ></rx-checkbox>
      <rx-checkbox v-model="checked6" label="Option 2" size="small" ></rx-checkbox>
    </div>
    <div>
      <rx-checkbox v-model="checked5" label="Option 1" size="small" disabled ></rx-checkbox>
      <rx-checkbox v-model="checked6" label="Option 2" size="small" disabled ></rx-checkbox>
    </div>
  </el-card>
</template>
<script setup lang="ts">
import { ref } from 'vue'
const checked1 = ref(true)
const checked2 = ref(false)
const checked3 = ref(false)
const checked4 = ref(false)
const checked5 = ref(false)
const checked6 = ref(false)
</script>

                                # 按钮样式

                                按钮样式的多选组合。

                                只需要把 rx-checkbox 元素替换为 rx-checkbox-button 元素即可。 此外，还提供了size属性。
                                ShanghaiBeijingGuangzhouShenzhen
                                ShanghaiBeijingGuangzhouShenzhen
                                ShanghaiBeijingGuangzhouShenzhen
                                ShanghaiBeijingGuangzhouShenzhen
                                按钮样式 demo
                                <template>
  <el-card shadow="never">
    <div>
      <rx-checkbox-group v-model="checkboxGroup1" size="large">
        <rx-checkbox-button v-for="city in cities" :key="city" :label="city">
          {{ city }}
        </rx-checkbox-button>
      </rx-checkbox-group>
    </div>
    <div class="demo-button-style">
      <rx-checkbox-group v-model="checkboxGroup2">
        <rx-checkbox-button v-for="city in cities" :key="city" :label="city">{{
          city
        }}</rx-checkbox-button>
      </rx-checkbox-group>
    </div>
    <div class="demo-button-style">
      <rx-checkbox-group v-model="checkboxGroup3" size="small">
        <rx-checkbox-button
          v-for="city in cities"
          :key="city"
          :label="city"
          :disabled="city === 'Beijing'"
          >{{ city }}</rx-checkbox-button
        >
      </rx-checkbox-group>
    </div>
    <div class="demo-button-style">
      <rx-checkbox-group v-model="checkboxGroup4" size="small" disabled>
        <rx-checkbox-button v-for="city in cities" :key="city" :label="city">{{
          city
        }}</rx-checkbox-button>
      </rx-checkbox-group>
    </div>
  </el-card>
</template>
<script lang="ts" setup>
import { ref } from 'vue'
const checkboxGroup1 = ref(['Shanghai'])
const checkboxGroup2 = ref(['Shanghai'])
const checkboxGroup3 = ref(['Shanghai'])
const checkboxGroup4 = ref(['Shanghai'])
const cities = ['Shanghai', 'Beijing', 'Guangzhou', 'Shenzhen']
</script>

<style scoped>
.demo-button-style {
  margin-top: 24px;
}
</style>

                                # 插槽

                                支持函数式插槽slots、模板插槽template

                                TIP

                                默认插槽为template模板插槽
                                函数式插槽slots渲染优先级大于模板插槽template

                                模板插槽Label
                                插槽 demo
                                <template>
  <el-card shadow="never">
    <!-- 模板插槽 -->
    <rx-checkbox v-model="checked1" label="模板插槽" size="large">
      <!-- $slots.default -->
      <div>模板插槽Label</div>
    </rx-checkbox>
    <!-- 模板插槽 -->
    <rx-checkbox v-model="checked2" label="函数式插槽" size="large" :slots="checkboxFncSlots" ></rx-checkbox>
  </el-card>
</template>

<script lang="ts" setup>
import { ref, h } from 'vue'
const checked1 = ref(false)
const checked2 = ref(false)
const checkboxFncSlots = () => h('div', {}, '函数式插槽Label')
</script>
